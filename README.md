# StudentOS

StudentOS is a modern productivity dashboard designed specifically for students.

It helps students manage their studies, tasks, exams, and focus sessions in one place.

The goal of StudentOS is to create a powerful study system that helps students stay organized, focused, and consistent.

---

## Features

- Task management system
- Subject progress tracking
- Study focus timer (Pomodoro)
- Upcoming exam tracker
- Clean modern dashboard UI
- Designed specifically for students

---

## Tech Stack

StudentOS is built with modern web technologies.

- Next.js
- React
- TypeScript
- Tailwind CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/kushhooda/studentos.git
```

Move into the project directory:

```bash
cd studentos
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the app in your browser:

http://localhost:3000
